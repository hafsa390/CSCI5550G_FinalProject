WILD RUN is a 2D left-to-right scrolling action platformer built in NESMaker
for the original NES hardware. The player, a field researcher named DAVID got
stuck in the Jungle. He must fight through two zones of increasingly complex
terrain to reach home. Small ghosts will appear in the first zone and Blob
monster would appear in the second zone. DAVID’s goal is to arrive home by
killing those monsters.
The Scrolling platformer module and the Mega-metro-vania module are used
to build this game, along with some edits in the assembly scripts. The ideas from
the intermediate modules of Scrolling platformer (jumping through platforms,
avoiding hazards) and Simple Shump(Blob monster) are implicitly adopted.
